## 1. 공통 컴포넌트 및 설계 원칙

### 1.1 반응형 브레이크포인트

| 구분 | 너비 | 레이아웃 |
|------|------|---------|
| Mobile | 360px | 단일 컬럼, 하단 탭 네비게이션 |
| Tablet | 768px | 2컬럼, 상단 네비게이션 전환 |
| Desktop | 1280px | 사이드바 + 콘텐츠 그리드 |

### 1.2 공통 Header

```
[Mobile 360px]
+------------------------------------------+
|  TodoApp                    [로그아웃]    |  <- 공통 Header (sticky, height: 56px)
|  안녕하세요, 홍길동님                     |  <- 로그인된 사용자명
+------------------------------------------+
|  [할일목록]     [카테고리]     [내정보]   |  <- 하단 탭 네비게이션
+------------------------------------------+

[Desktop 1280px]
+--------------------------------------------------------------------------------+
|  [TodoApp Logo]   [할일목록]  [+ 새 할일 등록]      [👤 홍길동 ▼]  [로그아웃] |
+--------------------------------------------------------------------------------+
                                                        ↓ 드롭다운
                                                +-------------------+
                                                |  내 계정 설정     |
                                                |  ─────────────    |
                                                |  로그아웃         |
                                                +-------------------+
```

### 1.3 공통 컴포넌트 상태 정의

```
[입력 필드 상태]
+----------------------------+   기본    (border: #D1D5DB)
| placeholder text           |
+----------------------------+

+============================+   포커스  (border: #3B82F6, ring: 2px #BFDBFE)
| 입력 중인 텍스트            |
+============================+

+----------------------------+   정상 완료 (border: #10B981)
| 유효한 입력값          [v] |
+----------------------------+

+- - - - - - - - - - - - - -+   에러    (border: #EF4444)
| 잘못된 입력값              |
+- - - - - - - - - - - - - -+
! 인라인 에러 메시지 (red, 12px)

+############################+   읽기 전용 / 비활성 (bg: #F3F4F6)
| 읽기 전용 값               |
+############################+


[버튼 상태]
[   버튼명   ]   기본     (bg: #3B82F6 / 텍스트 white)
[   버튼명   ]   Hover    (bg: #2563EB)
[ o 처리 중  ]   Loading  (Spinner + bg: #3B82F6 / opacity 0.85 / 클릭 불가)
[   버튼명   ]   Disabled (bg: #D1D5DB / 텍스트 #9CA3AF / 클릭 불가)
[   삭제     ]   Danger   (bg: #EF4444 / 텍스트 white)
[   취소     ]   Ghost    (bg: transparent / 테두리 #D1D5DB)
[ 버튼 ░░░  ]   Disabled 기본 카테고리 삭제 등 (opacity 0.4, cursor: not-allowed)


[Toast 컴포넌트]
+--------------------------------------------------+
| v  성공 메시지                              [x]  |   Success (bg: #F0FDF4 / 테두리: #10B981)
+--------------------------------------------------+

+--------------------------------------------------+
| !  에러 메시지                              [x]  |   Error   (bg: #FEF2F2 / 테두리: #EF4444)
+--------------------------------------------------+

+--------------------------------------------------+
| i  경고 / Rate Limit 안내                   [x]  |   Warning (bg: #FFFBEB / 테두리: #F59E0B)
+--------------------------------------------------+

- Mobile:  화면 하단 center, margin: 8px
- Desktop: 화면 우하단, margin: 16px
- 지속 시간: 3초 후 자동 소멸 + 수동 [x] 닫기


[Badge 컴포넌트]
[WORK]      파란색 배경, 흰색 텍스트
[PERSONAL]  초록색 배경, 흰색 텍스트
[STUDY]     보라색 배경, 흰색 텍스트
[HEALTH]    주황색 배경, 흰색 텍스트
[OVERDUE]   빨간색 배경, 흰색 텍스트  <- 클라이언트 로컬 타임존 기준 판단 (NFR-07)


[Modal 공통 규칙]
- 배경: rgba(0,0,0,0.5) 오버레이 dim 처리
- ESC 키 또는 배경 클릭 -> 닫기 (취소 동작)
- 포커스 트랩: 탭 이동이 Modal 내부에서만 순환 (접근성)
- 탈퇴 2단계 Modal: 배경 클릭 닫기 비활성화 (실수 방지)
```

## 4. 할일 목록 페이지 `/`

> **UC-08**: 카테고리/완료 여부/기간만료 필터 적용 → 목록 출력
> **UC-06**: 완료 토글 (Optimistic Update)  **UC-07**: 삭제 확인 후 제거

### Mobile (360px) — 기본 상태

```
+------------------------------------------+
|  TodoApp                    [로그아웃]    |
|  안녕하세요, 홍길동님                     |
+------------------------------------------+
|  [할일목록]     [카테고리]     [내정보]   |  <- 하단 탭 네비게이션
+------------------------------------------+
|                                          |
|  할일 목록                               |  <- 페이지 타이틀
|  [+ 새 할일 등록]                        |  <- Primary Button (full-width)
|                                          |
+------------------------------------------+
|  필터                                    |
|                                          |
|  카테고리                                |
|  +--------------------------------------+|
|  | 전체                         [▼]    ||  <- category select (URL: ?category_id=)
|  +--------------------------------------+|
|                                          |
|  [v] 완료 항목 포함                      |  <- checkbox (URL: ?is_completed=true)
|  [ ] 기간 만료 항목만                    |  <- checkbox (URL: ?expired=true)
|                                          |
+------------------------------------------+
|                                          |
|  +--------------------------------------+|
|  | [v] [WORK] [OVERDUE]                ||  <- 체크박스 + 카테고리 Badge + Overdue Badge
|  |  회의 자료 준비                      ||  <- title (필수)
|  |  2026-05-10 ~ 2026-05-12            ||  <- start_date ~ due_date
|  |  기획팀 Q2 회의 자료 작성...        ||  <- description (말줄임)
|  |                      [수정] [삭제]  ||  <- 액션 버튼
|  +--------------------------------------+|
|                                          |
|  +--------------------------------------+|
|  | [v] [PERSONAL]                      ||
|  |  독서 30분                           ||
|  |  2026-05-13 ~                       ||  <- due_date 없으면 start_date만
|  |                      [수정] [삭제]  ||
|  +--------------------------------------+|
|                                          |
|  +--------------------------------------+|
|  | [x] [STUDY] ~~완료된 항목~~         ||  <- is_completed=true: 취소선
|  |  ~~React 강의 수강~~                ||
|  |  2026-05-11 ~ 2026-05-13            ||
|  |                      [수정] [삭제]  ||
|  +--------------------------------------+|
|                                          |
|  [ o Loading... ]                        |  <- 추가 로딩 시 Spinner
|                                          |
+------------------------------------------+

[v] = 미완료 체크박스  (클릭 시 Optimistic Update 즉시 반영)

|              |  +--------------------------------------+  |
|              |                                            |
|              |  이 작업은 되돌릴 수 없습니다.            |  <- 보조 안내 (gray)
|              |                                            |
|              |--------------------------------------------|
|              |      [취소]              [삭제 확인]       |
|              |   <- secondary        <- Danger (red)      |

[취소]      -> Modal 닫기, 카테고리 유지

```

|              |  |   • 작성한 모든 할일 데이터          |  |
